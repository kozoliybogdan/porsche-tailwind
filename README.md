# ДЗ 15.1 — Tailwind CSS

Це односторінковий проєкт, який був перероблений з Bootstrap на **Tailwind CSS**.

## Технології
- Tailwind CSS через CDN
- Flowbite (готові Tailwind-компоненти + JS для dropdown/collapse/accordion/carousel)

## Запуск локально
Варіант 1 (найпростіший):
- Відкрийте файл `index.html` подвійним кліком.

Варіант 2 (рекомендовано, щоб все точно працювало як на GitHub Pages):
- Встановіть розширення **Live Server** у VS Code.
- Відкрийте проєкт у VS Code.
- Натисніть **Go Live**.

## Деплой на GitHub Pages
1. Запуште репозиторій на GitHub.
2. Settings → Pages → Deploy from branch.
3. Branch: `main`, Folder: `/ (root)`.

Після цього GitHub Pages видасть посилання на сайт.
