# ДЗ 15.1 — Tailwind CSS

Це односторінковий проєкт, який був перероблений з Bootstrap на **Tailwind CSS**.

## Технології
- Tailwind CSS через CDN
- Flowbite (готові Tailwind-компоненти + JS для dropdown/collapse/accordion/carousel)

## Запуск локально
- Відкрийте файл `index.html` подвійним кліком.
- АБО
- Встановіть розширення Live Server
